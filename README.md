# MCM 2026 Modeling Project

## Overview
This repository contains our Mathematical Contest in Modeling (MCM) project, developed to address a structured ranking and decision-analysis problem through optimization, Bayesian modeling, and simulation.

## Methods
The project combines:
- linear programming for feasible solution construction
- Bayesian modeling for latent structure and uncertainty
- simulation-based comparison under alternative mechanisms
- quantitative metrics for consistency, stability, and evaluation

## Outputs
The repository includes:
- the final competition paper
- core implementation code
- selected model outputs and figures
- the original problem statement

## Repository Structure
- `paper/` — final paper
- `code/` — modeling and analysis code
- `figures/` — charts and visual outputs
- `docs/` — original prompt and project materials
