# MCM Fan Voting Analysis

## Overview
This repository contains our Mathematical Contest in Modeling (MCM) project, which analyzes a structured fan-voting problem using quantitative modeling, simulation, and comparative evaluation.

## Methods
The project combines:
- optimization-based modeling
- Bayesian/statistical modeling
- simulation and scenario comparison
- quantitative evaluation metrics

## Files
- `main_model.py` — core modeling and analysis code
- `mcm-2026-finalpaper` — final competition paper
- `problem_statement.pdf` — original competition prompt

## Notes
This repository presents the project in a cleaner format for review, with core code and final written output included.
